# 情報パネルUI仕様

## 概要
画面右上に表示されるパネルで、選択されたエンティティの詳細情報を表示します。

## 表示対象

### ソウル選択時
- **ヘッダー**: 名前（または「Damned Soul」）
- **性別アイコン**: `male.jpg` / `female.jpg`
- **ステータス**:
  - Motivation: {パーセンテージ}
  - Stress: {パーセンテージ}（アイコン付き）
  - Fatigue: {パーセンテージ}（アイコン付き）
- **タスク**: 現在のタスク状態
- **所持品**: 運搬中のアイテム

### 使い魔選択時
- **ヘッダー**: 使い魔名
- **共通テキスト**: タイプ、範囲、疲労閾値

### その他のエンティティ
- Blueprint: タイプ、進行状況
- Resource Item: タイプ
- Tree/Rock: 自然資源情報

## 関連ファイル
- `src/interface/ui/panels.rs` - 情報パネルシステム
- `src/interface/ui/setup/panels.rs` - 初期UIセットアップ
