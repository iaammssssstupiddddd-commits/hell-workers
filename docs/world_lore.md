# Hell Workers - 世界観設定書

> アセットデザインのための世界観・視覚指針

---

## 1. 世界の概要

### 1.1 プロジェクト「永久建設」

地獄では今、史上最大の建設プロジェクトが進行中である。

地獄の王が、新たな「罪人大楼（シナーズ・タワー）」を建設するため、永劫に放置されてきた怠惰な魂たちを再編成した。これは、地獄の厳格な管理下での「建設シミュレーション」であり、魂たちは強制的に労働者として再教育されている。

### 1.2 舞台：第7建設区域「懶惰の野」

本作の舞台は、地獄の辺境に位置する第7建設区域「懶惰の野（らだのの）」である。

かつてこの土地は、怠惰の罪で堕ちてきた魂たちが永遠にだらだらと過ごす「無為の楽園」だった。しかし、建設計画の開始により、彼らは突然「労働者」へと身分を変えられた。

| 要素 | 設定 |
|:---|:---|
| **区域名** | 懶惰の野（Zone of Sloth） |
| **特徴** | 川が横断し、森林と岩山が混在する未開の地 |
| **気候** | 常に薄暗く、赤黒い空が広がる |
| **時間の流れ** | 人間界と異なり、遅延と停滞が常態化 |

## 2. 地獄の地理と環境

### 2.1 地形の特徴

```
┌─────────────────────────────────────────────────────────────┐
│  [Forest Zone]          [River]          [Rock Area]       │
│     🌲🌲🌲              ～～～              ⛰️⛰️⛰️        │
│     🌲🌲🌲            ～～～～～            ⛰️⛰️⛰️        │
│   (樹海の果て)       (忘却の川)          (悔恨の岩山)       │
│                                                             │
│              [Base Camp: 拠点]                             │
│                   🏗️                                        │
└─────────────────────────────────────────────────────────────┘
```

#### 森林エリア「樹海の果て」
- **由来**: 人間界で木を伐採しすぎた魂が堕ちてくる場所
- **特徴**: 伐採しても再生する呪いの木々が生い茂る
- **資源**: 木材（魂の肋骨を模した黒い木）

#### 忘却の川
- **由来**: 生前の記憶を流す川。地獄を横断している
- **特徴**: 黒く濁った水。渡河できない障害物
- **岸辺**: 砂浜（Sand）が広がり、魂たちはここで休息を取る

#### 悔恨の岩山
- **由来**: 頑固だった魂たちが岩になった場所
- **特徴**: 採掘すると土に変わる（通行可能になる）
- **資源**: 石材（魂の叫びがこもった灰色の岩）

### 2.2 環境エフェクト

| 現象 | 視覚表現 | 理由 |
|:---|:---|:---|
| **空** | 暗赤色のグラデーション | 地獄の「空」は常に夕暮れ時 |
| **光** | 松明や魂の微かな光 | 自然光がない |
| **霧** | 薄紫色の靄 | 怠惰の瘴気 |
| **パーティクル** | 灰のようなものが舞う | 燃え尽きた魂の残滓 |

## 3. 魂（Damned Soul）の生態

### 3.1 魂の起源

魂たちは、生前「怠惰（スロース）」の罪を犯した人間である。

- **名前**: 生前の名前（James, Mary等）を保持
- **外見**: 半透明の霊体。輪郭がぼんやりしている
- **色**: 基本は白〜青白い色だが、感情で変化

### 3.2 魂の感情と色彩

魂の体色は感情状態によって変化する。

| 感情 | 色 | 発生条件 |
|:---|:---|:---|
| **通常** | 白〜薄青 | デフォルト状態 |
| **やる気** | 黄緑 | タスク割当時 |
| **喜び** | ピンク | タスク完了時 |
| **疲労** | グレー | 疲労限界 |
| **ストレス** | 赤 | ストレス崩壊時 |
| **恐怖** | 紫 | リクルート・逃走時 |
| **安心** | 水色 | 解放時 |
| **リラックス** | ミント | 集会参加時 |

### 3.3 魂の社会構造

魂たちには、非公式な「サボり文化」が存在する。

#### 集会（Gathering Spot）
魂たちが集まって井戸端会議をする場所。

| 人数 | 中心オブジェクト | 意味 |
|:---|:---|:---|
| 2〜4人 | なし（座り込み） | 小規模な休息 |
| 5〜6人 | カードテーブル | 賭博（魂はお金を持たないが楽しむ） |
| 7〜8人 | 焚き火 | 大規模な休息と情報交換 |
| 7〜8人 | 樽（酒樽） | 虚構の「酒」を飲む（味はしない） |

**紫のオーラ**: 集会スポットには、怠惰の象徴として紫色のオーラが漂う。

#### 逃走文化

使い魔に使役されることを嫌い、自由な時間を求めて逃走する。

- **警戒圏**: 使い魔から一定距離に入るとストレス上昇
- **安全圏**: 集会スポットや使い魔から離れた場所で休息
- **逃走時の姿**: 体が青白くなり、傾いた姿勢で高速移動

### 3.4 魂の言語

魂は言葉を話さず、感情を「絵文字」で表現する。

| 絵文字 | 意味 | 使用場面 |
|:---|:---|:---|
| 💪 | やる気 | タスク開始 |
| 😊 | 満足 | タスク完了 |
| 😴 | 疲労 | 限界到達 |
| 😰 | パニック | ストレス崩壊 |
| 😨 | 恐怖 | リクルート時 |
| 😅 | 安心 | 解放時 |
| 😌 | リラックス | 集会参加 |
| 😓 | 不満 | タスク放棄 |
| 💤 | 退屈 | 長時間アイドル |

## 4. 使い魔（Familiar）の階級

### 4.1 使い魔の役割

使い魔は地獄の「下級官吏」であり、魂たちの監督官である。

- **外見**: 小さな悪魔のような生物。翼と尻尾を持つ
- **サイズ**: 魂より小さい（威圧感を抑えるため）
- **目**: 鋭い眼光。複数の瞳を持つものもいる

### 4.2 使い魔の階級

| 階級 | 管理人数 | 特徴 |
|:---|:---:|:---|
| **小鬼（Imp）** | 1〜2名 | 新人。まだ威厳がない |
| **従魔（Servitor）** | 3〜4名 | 一般的な監督官。赤い肌 |
| **上級魔（Greater）** | 5〜6名 | ベテラン。青い炎を纏う |
| **統率魔（Overseer）** | 7〜8名 | 最上位。金色の装飾を持つ |

### 4.3 使い魔の言語：古代ラテン語

使い魔は命令を「古代ラテン語」で発する。これは地獄の伝統である。

| ラテン語 | 意味 | 使用場面 |
|:---|:---|:---|
| Caede | 切れ、伐採せよ | 伐採命令 |
| Fodere | 掘れ、採掘せよ | 採掘命令 |
| Portare | 運べ | 運搬命令 |
| Laborare | 働け、構築せよ | 建築命令 |
| Abi | 去れ | 解放命令 |
| Veni | 来たれ | リクルート時 |
| Requiesce | 休め | 待機命令 |

**吹き出し**: 使い魔のセリフは「9スライス吹き出し」にタイプライター効果で表示される。

### 4.4 使い魔の能力

| 能力 | 効果 | 視覚表現 |
|:---|:---|:---|
| **激励** | 魂のやる気回復 | 🔥や⚡のエフェクト |
| **監視光線** | 威圧的な視線 | 目から赤い光線 |
| **分隊統率** | 複数魂の管理 | 魂と使い魔を結ぶ線 |

## 5. 建築と資材

### 5.1 設計図（Blueprint）

建築は「青白く光る設計図」から始まる。

- **外見**: 半透明な青い投影
- **状態表示**:
  - 資材不足時: 赤みがかる
  - 準備中: 黄色く光る
  - 建築可能: 緑色に変化
  - 建築中: 青色で脈動

### 5.2 資材の特徴

| 資材 | 由来 | 外見 |
|:---|:---|:---|
| **木材** | 魂の肋骨を模した黒い木 | 暗い茶色、白い筋 |
| **石材** | 叫び声がこもった悔恨の岩 | 灰色、微かな光 |
| **水** | 忘却の川の水 | 黒く濁った液体 |
| **骨** | 怠惰な魂の成れの果て | 白く乾いた骨片 |

### 5.3 建築物

| 建築物 | 用途 | デザイン |
|:---|:---|:---|
| **壁（Wall）** | 区域の区切り | 黒い石積み |
| **床（Floor）** | 通路の舗装 | 石畳 |
| **備蓄場（Stockpile）** | 資材保管 | 紫色のオーラを放つ領域 |
| **水タンク** | 水の貯蔵 | 骨組みのタンク |

### 6.2 アートスタイル仕様（Rough Vector Sketch）

本作のアートスタイルは、**「手描き感の強いラフなベクターイラスト」**で統一する。
「完成されすぎた綺麗な絵」ではなく、開発中のスケッチのようなアナログ感を意図的に残す。

#### ラインワークと塗り
| 要素 | 特徴 |
|:---|:---|
| **輪郭線** | 太く、インクが滲んだようなルーズさ（Loose lines）と揺らぎ（Wobbly）を持たせる |
| **塗り** | 均一ではなく、筆跡や塗りムラ（Textured brush）を残し、少し未完成な味わいを出す |
| **情報量** | 描き込みすぎず、シルエットを重視したシンプルな構成 |

#### 視点と構図
| ルール | 詳細 |
|:---|:---|
| **正投影** | 完全なOrthographic Projection（平行投影）で描く。3/4 viewは禁止 |
| **配置** | 壁や配管などの連結物は必ず水平または垂直に真っ直ぐ配置。斜めの配置は禁止 |
| **立体感** | 壁などは正面（Front face）と上面（Top edge）が見える角度で描画 |

#### 形状の特徴
| 要素 | 特徴 |
|:---|:---|
| **全体** | ティム・バートン作品のように、少し歪んだり傾いたりしているシルエット |
| **木材** | 魂の肋骨のような質感。黒く歪んでいる |
| **裂け目** | 紫の光（Purple Glow）が漏れる「怠惰のエネルギー」 |
| **金属** | 錆びた鉄（Rusty Metal）。トゲのあるバンドや補強パーツ |

### 6.3 アニメーション指針

| アクション | アニメーション | 備考 |
|:---|:---|:---|
| **魂の移動** | ふわふわ浮遊 | 物理的な歩行ではない |
| **使い魔の移動** | 羽ばたき飛行 | 小刻みに上下 |
| **建築** | ハンマーアイコン | 魂の頭上に表示 |
| **採取** | ツールの振り下ろし | プログレスバー同期 |
| **逃走** | 高速移動＋点滅 | 青白い色で傾斜 |

## 7. ストーリー的要素

### 7.1 建設プロジェクトの目的

「罪人大楼（シナーズ・タワー）」は、次の大罪人的な魂を収容するための新施設である。

- **現在の地獄**: 収容人数が限界に近づいている
- **王の計画**: さらに多くの魂を効率的に「処理」する施設
- **皮肉**: 怠惰の罪人被せて建設を行わせている

### 7.2 魂たちの心境

- **生前**: 怠け者で、何もせず時間を浪費
- **地獄での生活**: 永遠のだらだらを期待していた
- **現在**: 強制労働に驚きと不満
- **抵抗**: サボり、逃走、集会を通じて怠惰を維持しようとする

### 7.3 使い魔たちの本音

- **表向き**: 厳格な監督官
- **内心**: 魂たちの怠惰に共感することも（昔は人間だった頃の記憶が…）
- **激励**: 魂たちを気遣いながらも、命令は命令として実行
